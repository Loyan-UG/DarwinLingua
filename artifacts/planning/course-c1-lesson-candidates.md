# C1 Course Lesson Candidates

Date: 2026-05-31

## Purpose

This file preserves the reviewed C1 `CourseLesson` backlog before full multilingual course content generation.

Use it to keep C1 Course content comparable to advanced German coursebooks, Goethe C1, telc Deutsch C1 Hochschule, academic German, professional strategy communication, and nuanced public/social discourse while staying compatible with Darwin Lingua's German-first, linked-content architecture.

## Reference Anchors

The C1 plan aligns with:

- CEFR C1 expectations: fluent, spontaneous, flexible, and effective language use for social, academic, and professional purposes; clear, well-structured, detailed expression on complex subjects; controlled use of organization, connectors, and cohesive devices.
- Goethe/telc C1 needs: advanced reading, listening, writing, and speaking; academic/professional input; presentation and discussion; mediation-like summary tasks; free writing; and language elements at advanced level.
- German academic/professional needs: seminar discussion, source evaluation, structured argument, formal style, strategic meetings, complex complaint/escalation, and careful register shifting.
- Existing Darwin Lingua assets: C1 Grammar topics, C1 RoleplayScenario content, C1 Expressions, B2 Course bridge, Talk Topics, Dialogues, Exercise sets, and future Exam Prep links.

## C1 Generation Rules

- Source `title`, `description`, `shortDescription`, `narrative`, `learningGoals`, `reviewSummary`, and `homeworkTask` must be German-first.
- Add learner-helper translations for `en`, `fa`, `ar`, `tr`, `ru`, `ckb`, `kmr`, `pl`, `ro`, and `sq`.
- C1 lessons should use dense but readable German, authentic discourse patterns, careful argument structure, and clear explanations of style/register.
- Each lesson should normally include 5-6 concise German learning goals.
- C1 should train abstraction, nuance, concession, hedging, source criticism, reported speech, mediation, strategic workplace communication, and academic/professional writing.
- Course lessons must link to existing content where useful, but must not copy full Grammar/Expression/Dialogues/Roleplay/Exercise content.
- `linkedWordSlugs` may stay empty until a stable server-side word-slug convention exists.
- No sensitive/adult content belongs in this general C1 course path.
- Course imports for an existing path are cumulative: every package update must keep all reviewed lessons produced so far.
- Content production should start only after A1-C2 Course lesson planning files exist and are reviewed.

## Production Readiness Gate

Before generating full C1 lesson content from this plan:

- Confirm A1, A2, B1, B2, C1, and C2 Course planning files exist.
- Confirm the user explicitly starts content generation after planning is complete.
- Use Plan mode for the first content-generation prompt of each level.
- Generate only a small cumulative batch at a time.
- Keep all German source fields German-first and add learner-helper translations for all active learner languages.
- Require C1-level depth: dense but readable German, nuanced argumentation, careful register control, and authentic academic/professional discourse.
- Do not produce generic textbook filler; each lesson must include a concrete communicative or academic/professional outcome.
- Run JSON/language preflight and Course parser/import validation for each batch.
- Import to `darwinlingua_shared` only after validation.
- Verify PostgreSQL counts after import.
- Run lightweight Web/API smoke for at least one new lesson per batch.
- Take a database backup at the end of each completed CEFR level, not after every small batch.
