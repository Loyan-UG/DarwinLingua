# Exam Prep C1 Title Candidates

## Purpose

Reviewed C1 Exam Prep candidate backlog for the first C1 foundation package. Titles must be clean learner tasks; provider, CEFR level, section, and task type stay in metadata and filters.

## Generation Rules

- Source titles, descriptions, explanations, strategy notes, and checklists are German-first.
- Helper translations cover `en`, `fa`, `ar`, `tr`, `ru`, `ckb`, `kmr`, `pl`, `ro`, and `sq`.
- Do not copy official provider tasks, prompts, transcripts, answer keys, or sample exam material.
- Link only imported and reviewed slugs. C1 Exercise and Writing Template links stay empty until real content exists.
- Prefer C1 strategies: synthesis, nuance, register control, argument structure, source handling, academic speech, and formal writing.
