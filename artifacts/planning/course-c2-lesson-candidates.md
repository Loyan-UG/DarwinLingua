# C2 Course Lesson Candidates

Date: 2026-05-31

## Purpose

This file preserves the reviewed C2 `CourseLesson` backlog before full multilingual course content generation.

Use it to keep C2 Course content comparable to the highest-level German coursebooks, Goethe-Zertifikat C2 / Grosses Deutsches Sprachdiplom preparation, advanced academic/professional discourse, rhetorical style work, culturally dense communication, and demanding public-language comprehension while staying compatible with Darwin Lingua's German-first, linked-content architecture.

## Reference Anchors

The C2 plan aligns with:

- CEFR C2 expectations: effortless comprehension of virtually everything heard or read, summarizing from different spoken and written sources, reconstructing arguments and accounts coherently, and expressing oneself spontaneously, very fluently, and precisely with fine shades of meaning.
- Goethe/telc-style C2 needs: complex reading, listening, writing, and speaking; literary/cultural and factual argumentation; advanced text production; nuanced discussion; and precise response under pressure.
- Advanced German-life needs: expert register, public debate, academic/technical discussion, strategic crisis communication, subtle pragmatic meaning, irony, ambiguity, culturally loaded language, and refined editing.
- Existing Darwin Lingua assets: C2 Grammar topics, C2 RoleplayScenario content, C2 Expressions, C1 Course bridge, Talk Topics, Dialogues, Exercise sets, and future Exam Prep links.

## C2 Generation Rules

- Source `title`, `description`, `shortDescription`, `narrative`, `learningGoals`, `reviewSummary`, and `homeworkTask` must be German-first.
- Add learner-helper translations for `en`, `fa`, `ar`, `tr`, `ru`, `ckb`, `kmr`, `pl`, `ro`, and `sq`.
- C2 lessons should use demanding but pedagogically clear German, with explicit attention to style, connotation, ambiguity, register, rhythm, rhetoric, and discourse strategy.
- Each lesson should normally include 5-7 concise German learning goals.
- C2 should train synthesis, stylistic variation, implicit meaning, expert-text comprehension, public debate, literary/cultural interpretation, crisis communication, and refined self-editing.
- Course lessons must link to existing content where useful, but must not copy full Grammar/Expression/Dialogues/Roleplay/Exercise content.
- `linkedWordSlugs` may stay empty until a stable server-side word-slug convention exists.
- No sensitive/adult content belongs in this general C2 course path.
- Course imports for an existing path are cumulative: every package update must keep all reviewed lessons produced so far.
- Content production should start only after A1-C2 Course lesson planning files exist and are reviewed.

## Production Readiness Gate

Before generating full C2 lesson content from this plan:

- Confirm A1, A2, B1, B2, C1, and C2 Course planning files exist.
- Confirm the user explicitly starts content generation after planning is complete.
- Use Plan mode for the first content-generation prompt of each level.
- Generate only a small cumulative batch at a time.
- Keep all German source fields German-first and add learner-helper translations for all active learner languages.
- Require C2-level depth: stylistic choice, implicit meaning, synthesis, ambiguity control, register precision, and authentic high-level discourse.
- Do not produce generic textbook filler; each lesson must include a concrete expert, academic, professional, social, or public-discourse outcome.
- Run JSON/language preflight and Course parser/import validation for each batch.
- Import to `darwinlingua_shared` only after validation.
- Verify PostgreSQL counts after import.
- Run lightweight Web/API smoke for at least one new lesson per batch.
- Take a database backup at the end of each completed CEFR level, not after every small batch.
