# B1 Course Lesson Candidates

Date: 2026-05-31

## Purpose

This file preserves the reviewed B1 `CourseLesson` backlog before full multilingual course content generation.

Use it to keep B1 Course content comparable to common modern German coursebooks, integration-course progression, workplace communication needs, and Goethe/telc/DTZ-style B1 exam preparation while staying compatible with Darwin Lingua's German-first, linked-content architecture.

## Reference Anchors

The B1 plan aligns with:

- CEFR B1 expectations: connected speech on familiar topics, main points in clear standard language, experiences, events, reasons, explanations, plans, opinions, and practical problem-solving.
- Goethe/telc-style B1 exam needs: reading, listening, writing, and speaking modules; personal and formal emails/letters; opinion posts; paired planning; short presentation; responding to questions; everyday and professional situations.
- German integration and practical-life needs: public offices, housing, health, work, school/kindergarten, contracts, service complaints, transport, social participation, and digital everyday tasks.
- Existing Darwin Lingua assets: B1 Grammar topics, B1 RoleplayScenario content, B1 Expressions, A1-A2 Exercise foundation, Talk Topics, Dialogues, and future Exam Prep links.

## B1 Generation Rules

- Source `title`, `description`, `shortDescription`, `narrative`, `learningGoals`, `reviewSummary`, and `homeworkTask` must be German-first.
- Add learner-helper translations for `en`, `fa`, `ar`, `tr`, `ru`, `ckb`, `kmr`, `pl`, `ro`, and `sq`.
- B1 lessons should be more complete than A2: longer connected examples, clearer discourse markers, practical explanations, and integrated speaking/writing tasks.
- Each lesson should normally include 4-5 concise German learning goals.
- B1 should train reasons, consequences, alternatives, polite disagreement, structured storytelling, formal/semi-formal writing, and problem explanation.
- Course lessons must link to existing content where useful, but must not copy full Grammar/Expression/Dialogues/Roleplay/Exercise content.
- `linkedWordSlugs` may stay empty until a stable server-side word-slug convention exists.
- No sensitive/adult content belongs in this general B1 course path.
- Course imports for an existing path are cumulative: every package update must keep all reviewed lessons produced so far.
- Content production should start only after A1-C2 Course lesson planning files exist and are reviewed.

## Production Readiness Gate

Before generating full B1 lesson content from this plan:

- Confirm A1, A2, B1, B2, C1, and C2 Course planning files exist.
- Confirm the user explicitly starts content generation after planning is complete.
- Use Plan mode for the first content-generation prompt of each level.
- Generate only a small cumulative batch at a time.
- Keep all German source fields German-first and add learner-helper translations for all active learner languages.
- Do not produce generic textbook filler; each lesson must include a concrete communicative outcome.
- Run JSON/language preflight and Course parser/import validation for each batch.
- Import to `darwinlingua_shared` only after validation.
- Verify PostgreSQL counts after import.
- Run lightweight Web/API smoke for at least one new lesson per batch.
- Take a database backup at the end of each completed CEFR level, not after every small batch.
